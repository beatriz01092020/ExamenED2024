git branch: Muestra la rama en donde estas

git status: Muestra el status del git (Cambios realizados estado de los repositorios etc...)

git init: Inicializa el repositorio local en donde te encuentras

git commit -m "comentario": Guarda los cambios hechos en el repositorio local junto con un comentario de retroalimentación

git push: Empuja el contenido del repositorio local al repositorio remoto (lo sube)

git pull: Lleva los cambios hechos en el respositorio remoto al local

git clone "NombreSSH" Crea una copia del repositorio remoto en local lista para ser usada

git add . Añade los cambios realizados en el repositorio local "Estado de preparacion"

git checkout "argumento" Cambia a la rama seleccionada entre las comillas

git merge "rama de origen" "rama de destino" Fusiona el contenido de las ramas en una sola si estas en una rama no hace falta poner la de origen ya que se usara la rama en la que te encuentres

git branch - r "rama" Comando que se encarga de borrar una rama

git diff Comando que compara las diferencias de codigo entre una rama y otra.
