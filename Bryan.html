Comandos de Git

git init: Donde se quiere iniciar

git add "" o: Añadir archivos editados (su nombre si solo es un archivo o un ponto si van a ser todos)

git commit -m Nombre: Añadir los archivos editados a un guardado entre el git local y el git remoto

git push: Subir los datos guardados en el git local al remoto

git pull: Incorporar nuevos archivos que se hayan implementado en el repositorio

git branch: Crear una nuaeva rama aparte de la principal

git checkout: Entrar en una rama

git merge: Combinar la rama en la que estamos con la principal

git branch rm nombre: Borrar rama

git rm nombre: Borrar archivos del git remoto
