Comandos de Github

git init: para inicializar un directorio existente a un repositorio de GitHub.

git clone: para clonar un repositorio de GitHub en tu ordenador.

git add ficheros: para añadir ficheros.

git commit -m "mensaje": para confirmar los ficheros que has añadido.

git push: para subir los cambios al directorio remoto.

git pull: para bajarte los cambios al directorio local.

git branch nombre: para crear ramas.

git checkout nombre: para cambiar de rama.

git diff rama rama: para ver las diferencias entre dos ramas.

git merge rama rama: para fusionar dos ramas.
