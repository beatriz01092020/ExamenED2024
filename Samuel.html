Examen de Git 12/11/2024

git clone - para copiar el repositorio de github.com
git branch - para crear una rama
git add . - para agregar a git todos los archivos que cree
git status - para saber si los archivos han sido agragados correctamente
git commit -m "mensaje cualquiera" - para que los cambios realizados queden constancia de cuando y cual es su modificación
git push - para subir los cambios y guardarlos en el repositorio de github
git merge - para fusionar las ramas que hemos creado
git branch -r "nombre de la rama" - para eliminar las ramas creadas anteriormente
